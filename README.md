# Let learn C!

## Learning ?

1. Clone this repository
2. Choose a sample
3. Modify the code
4. Compile it and run
5. Goto step 2

## Contributing ?

1. Fork
2. Modify and commit
3. Send pull request !

## 如何學習？
1. Clone一份專案
2. 選一個範例檔案
3. 修改程式碼
4. 編譯並執行
5. 重複 (2)

## 如何貢獻？

1. Fork一份
2. 修改並且送Commit
3. 送Pull Request
